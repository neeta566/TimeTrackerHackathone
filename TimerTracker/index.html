Timer Tracker is used to focus on work.It is based on pomodoro timer.some features are customizable.

customizable Setting TimerTracker features :

Dark Mode: Click on Dark mode to switch between dark and light mode.

Alarm: Checked for Alarm on/off.

Work Session:increase work time and Work session accourdinig to task and for stop timer after a fixed time.

Long Break set a Number of work session for Long Break.

work Cycle: Set a work Cycle for long break. Alarm: Set Alarm form setting.

What is TimeTracker?
TimerTracker app is very helpful to work and study. The technique uses a timer to break down work into work time intervals, traditionally 25 minutes in length, separated by short breaks. Each interval is known as a work session.
What is Work cycle?
Work Cycle is Decided a Long Break after a fixed number of work session.
How to use TimeTracker?

Work Session: Add estimated work session. one work session is 25 minutes with short break of 5 minutes. Automatically Timer is off after a Fixed work session complete.

Work Cycle: Add estimated work Cycle for Long Break.Long break will running after a fixed work session (like after 2 and 4 session).

Start Timer: Start Timer and you have automatically get short break for 5 minutes and the alarm ring.

Fixed Work Session:Timer stop after a Fixed Work session automatically and ring alarm.

Alarm :Alarm is on when work session end and short break start.
